Discovering Global Climbing Communities

About

This project is designed to identify groups of climbers on a global scale. By storing data about climbers and their ascents, a visual map is created to demonstrate how these small groups form of climbers of similar abilities, tastes in mountains, tastes in regions. This allows us to see previously hidden connections amongst some of the worlds greatest mountaineers.

The top left controls allow you to change clustering variables, allowing the data to be presented in multiple ways. Try changing the clustering to 'range'.
